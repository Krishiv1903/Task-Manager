# 📝 Task Sync

🔗 **Live Demo:** https://tasksync-team.netlify.app/

A full-stack **Task Management Web Application** that helps users efficiently organize, track, and manage their daily tasks. Built with a modern tech stack, this app focuses on productivity, clean UI, and seamless user experience.

---

## 📌 Features

1. ✅ **Task Management**

   * Create, update, and delete tasks
   * Mark tasks as completed or pending

2. 🔐 **User Authentication**

   * Secure login and signup functionality
   * Personalized task management

3. 📊 **Dashboard View**

   * Overview of all tasks
   * Organized and user-friendly interface

4. ⚡ **Real-Time Interaction**

   * Instant updates without page reload

5. 📱 **Responsive Design**

   * Works smoothly across devices

---

## 🛠️ Tech Stack

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Deployment:**

  * Frontend deployed using **Netlify**
  * Backend deployed using **Render**

---

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Krishiv1903/Task-Manager.git
```

2. Navigate to the project directory

```bash
cd Task-Manager
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```
